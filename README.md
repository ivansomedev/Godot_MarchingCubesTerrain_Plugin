# Voxel Terrain Plugin for Godot 4

Marching Cubes terrain generator with non-destructive editing.

## Features
- 🗺️ Heightmap-based terrain generation
- 🕳️ Non-destructive editing with subtractive/additive modifiers
- 🏔️ Dome-shaped caves with parabolic geometry
- 🔗 Tunnel interpolation between connected domes

## Quick Start
1. Add a `Chunk` node to your scene
2. Set chunk size and cell size
3. Add `Dome` nodes as children
4. Press "March map regen"!

Unfortunately, the interface isn't yet finished. If something isn't working, try enabling or disabling the plugin in the project settings.
